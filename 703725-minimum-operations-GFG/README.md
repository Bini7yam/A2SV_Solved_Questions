# [Minimum Operations](https://www.geeksforgeeks.org/problems/find-optimum-operation4504/1)
## Easy
Given a number n. Find the minimum number of operations required to reach n starting from 0. You have two operations available:

Double the number 
Add one to the number

Example 1:
Input: n = 8
Output: 4
Explanation: 0 + 1 = 1 --&gt; 1 + 1 = 2 --&gt; 2 * 2 = 4 --&gt; 4 * 2 = 8.

Example 2:
Input: n = 7
Output: 5
Explanation: 0 + 1 = 1 --&gt; 1 + 1 = 2 --&gt; 1 + 2 = 3 --&gt; 3 * 2 = 6 --&gt; 6 + 1 = 7.
Constraints:1 &lt;= n &lt;= 106
